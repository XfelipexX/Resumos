📅 Guia Completo: Datas em Java (java.time)

🔷 1. Principais Classes

  • LocalDate → Data (ano, mês, dia)
  • LocalTime → Hora
  • LocalDateTime → Data + Hora
  • Instant → Momento global (UTC)

import java.time.LocalDate;
import java.time.LocalDateTime;
import java.time.LocalTime;

LocalDate data = LocalDate.now();
LocalTime hora = LocalTime.now();
LocalDateTime dataHora = LocalDateTime.now();

🔷 2. Criando Datas


LocalDate data = LocalDate.of(2025, 4, 17);
LocalDateTime dataHora = LocalDateTime.of(2025, 4, 17, 14, 30);

🔷 3. Convertendo String para Data


import java.time.LocalDate;
import java.time.format.DateTimeFormatter;

DateTimeFormatter formato = DateTimeFormatter.ofPattern("dd/MM/yyyy");

LocalDate data = LocalDate.parse("17/04/2025", formato);

🔷 4. Convertendo Data para String


String dataFormatada = data.format(formato);
System.out.println(dataFormatada);

🔷 5. Operações com Datas


LocalDate hoje = LocalDate.now();

LocalDate ontem = hoje.minusDays(1);
LocalDate amanha = hoje.plusDays(1);

LocalDate proximoMes = hoje.plusMonths(1);
LocalDate anoPassado = hoje.minusYears(1);

🔷 6. Diferença entre Datas


import java.time.Period;

LocalDate inicio = LocalDate.of(2024, 1, 1);
LocalDate fim = LocalDate.now();

Period periodo = Period.between(inicio, fim);

System.out.println(periodo.getYears() + " anos");

🔷 7. Trabalhando com Instant (Tempo Global)


import java.time.Instant;

Instant agora = Instant.now();

Instant passado = agora.minusSeconds(3600); // 1 hora atrás

🔷 8. Conversão com TimeZone


import java.time.Instant;
import java.time.LocalDateTime;
import java.time.ZoneId;

Instant instant = Instant.now();

LocalDateTime dataLocal = LocalDateTime.ofInstant(
    instant, ZoneId.systemDefault()
);

🔷 9. Resumo Geral

  • LocalDate → Data sem hora
  • LocalTime → Hora sem data
  • LocalDateTime → Data + hora
  • Instant → Tempo global (UTC)
  • DateTimeFormatter → Formatação

💡 Dica Final

Prefira sempre usar java.time ao invés de classes antigas como Date e Calendar.
