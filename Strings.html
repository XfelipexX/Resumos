Formatação de Strings

CRIAÇÃO / CONVERSÃO
String s = "Texto";Cria string
String.valueOf(10)Converte para string
Integer.parseInt("10")String para int
Double.parseDouble("10.5")String para double
s.toCharArray()String para char[]
s.getBytes()String para bytes
TAMANHO / ESTADO
s.length()Tamanho da string
s.isEmpty()Verifica vazia
s.isBlank()Vazia ou espaços
COMPARAÇÃO
s.equals("x")Compara conteúdo
s.equalsIgnoreCase("x")Ignora maiúsc/minúsc
s.compareTo("x")Ordem alfabética
s.compareToIgnoreCase("x")Ordem ignorando case
BUSCA / VERIFICAÇÃO
s.contains("x")Contém texto
s.startsWith("x")Começa com
s.endsWith("x")Termina com
s.matches("regex")Valida padrão
POSIÇÃO
s.indexOf("x")Primeira posição
s.lastIndexOf("x")Última posição
EXTRAÇÃO
s.charAt(0)Pega caractere
s.substring(1)Corta a partir
s.substring(1,4)Corta intervalo
s.subSequence(1,4)Parte da string
MODIFICAÇÃO
s.replace("a","b")Substitui
s.replaceAll("a","b")Substitui com regex
s.replaceFirst("a","b")Primeira ocorrência
FORMATAÇÃO
s.toUpperCase()Maiúsculo
s.toLowerCase()Minúsculo
s.trim()Remove espaços
s.strip()Remove espaços Unicode
s.stripLeading()Remove início
s.stripTrailing()Remove final
s.indent(2)Adiciona indentação
JUNÇÃO / DIVISÃO
s.concat("abc")Concatena
s + "abc"Concatenação simples
String.join("-", "a","b")Junta strings
s.split(",")Divide string
s.lines()Divide por linhas
OUTROS
s.repeat(3)Repete string
s.intern()Otimiza memória
s.translateEscapes()Interpreta escapes
