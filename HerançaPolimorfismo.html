📘 Guia Completo: Herança, Polimorfismo, Upcasting e Downcasting

🔷 1. Herança

Herança permite que uma classe (subclasse) herde atributos e métodos de outra (superclasse).

  • Reutilização de código
  • Organização hierárquica
  • Facilidade de manutenção

class Produto {
    String nome;
    double preco;

    public double precoFinal() {
        return preco;
    }
}

class ProdutoImportado extends Produto {
    double taxaImportacao;

    @Override
    public double precoFinal() {
        return preco + taxaImportacao;
    }
}

🔷 2. Polimorfismo

Permite que um método tenha comportamentos diferentes dependendo do objeto. 


Produto p1 = new Produto();
Produto p2 = new ProdutoImportado();

System.out.println(p1.precoFinal());
System.out.println(p2.precoFinal());

🔷 3. Upcasting

Conversão de subclasse para superclasse.

  • Automático
  • Seguro

Produto p = new ProdutoImportado();

🔷 4. Downcasting

Conversão de superclasse para subclasse.

  • Precisa de cast explícito
  • Pode gerar erro (ClassCastException)

Produto p = new ProdutoImportado();
ProdutoImportado pi = (ProdutoImportado) p;

Forma segura:


if (p instanceof ProdutoImportado) {
    ProdutoImportado pi = (ProdutoImportado) p;
}

🔷 5. Exemplo Completo


import java.util.ArrayList;
import java.util.List;

class Produto {
    String nome;
    double preco;

    public Produto(String nome, double preco) {
        this.nome = nome;
        this.preco = preco;
    }

    public double precoFinal() {
        return preco;
    }
}

class ProdutoImportado extends Produto {
    double taxaImportacao;

    public ProdutoImportado(String nome, double preco, double taxaImportacao) {
        super(nome, preco);
        this.taxaImportacao = taxaImportacao;
    }

    @Override
    public double precoFinal() {
        return preco + taxaImportacao;
    }
}

public class Main {
    public static void main(String[] args) {

        List<Produto> lista = new ArrayList<>();

        lista.add(new Produto("Teclado", 100));
        lista.add(new ProdutoImportado("Notebook", 2000, 300));

        for (Produto p : lista) {
            System.out.println(p.precoFinal());
        }
    }
}

🔷 6. Resumo Geral

  • Herança: Uma classe herda de outra
  • Polimorfismo: Métodos com comportamentos diferentes
  • Upcasting: Subclasse → Superclasse
  • Downcasting: Superclasse → Subclasse
