Guia Completo: Pacotes e Arquivos em Java

📁 Guia Completo: Pacotes e Manipulação de Arquivos em Java

🔷 1. Pacotes (package)

Pacotes servem para organizar classes em Java.


package com.meuprojeto.model;

Importando classes:


import java.util.Scanner;
import java.io.File;

🔷 2. Classe File (caminho de arquivo)


String caminho = "C:\\temp\\arquivo.txt";

File file = new File(caminho);

Representa arquivos ou diretórios.

🔷 3. Leitura de Arquivo


FileReader fr = null;
BufferedReader br = null;

try {
    fr = new FileReader(caminho);
    br = new BufferedReader(fr);

    String linha = br.readLine();

    while (linha != null) {
        System.out.println(linha);
        linha = br.readLine();
    }

} catch (IOException e) {
    System.out.println("Erro: " + e.getMessage());
}

🔷 4. Escrita de Arquivo


FileWriter fw = null;
BufferedWriter bw = null;

try {
    fw = new FileWriter(caminho);
    bw = new BufferedWriter(fw);

    bw.write("Linha 1");
    bw.newLine();
    bw.write("Linha 2");

} catch (IOException e) {
    System.out.println("Erro: " + e.getMessage());
}

🔷 5. Try-with-resources (Melhor prática)


try (BufferedWriter bw = new BufferedWriter(new FileWriter(caminho))) {
    bw.write("Texto automático");
    bw.newLine();
}

Fecha automaticamente os recursos.

🔷 6. Acrescentar conteúdo no arquivo


new FileWriter(caminho, true);

true → adiciona sem apagar o conteúdo existente

🔷 7. Listando arquivos e pastas


File path = new File("C:\\temp");

File[] files = path.listFiles(File::isFile);
File[] folders = path.listFiles(File::isDirectory);

🔷 8. Criando pastas


boolean sucesso = new File("C:\\temp\\subdir").mkdir();

🔷 9. Resumo Geral

  • File → representa arquivo/caminho
  • FileReader → leitura de caracteres
  • BufferedReader → leitura eficiente
  • FileWriter → escrita
  • BufferedWriter → escrita eficiente
  • readLine() → lê linha
  • write() → escreve
  • newLine() → quebra de linha

💡 Dica Final

Sempre prefira BufferedReader e BufferedWriter para melhor performance.
