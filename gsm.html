📦 Guia Completo: Generics, Set e Map em Java

🔷 1. Generics

Permite criar classes e coleções com tipos parametrizados.


List<String> lista = new ArrayList<>();
ParteExplicação
List<String>Lista que aceita apenas String
ArrayList<>Implementação da lista

Classe genérica:


public class Caixa<T> {
    private T valor;

    public void setValor(T valor) {
        this.valor = valor;
    }

    public T getValor() {
        return valor;
    }
}
ParteExplicação
<T>Tipo genérico (pode ser qualquer tipo)
setValor()Define o valor
getValor()Retorna o valor

Com restrição:


public class Exemplo<T extends Number> {
}
ParteExplicação
T extends NumberSó aceita números (Integer, Double...)

🔷 2. Coringa (Wildcard)

Representa qualquer tipo:


List<?> lista = new ArrayList<Object>();
TipoExplicação
?Qualquer tipo
? extends TSó leitura (subtipos)
? super TPermite escrita
  • ? extends T → leitura (subtipos)
  • ? super T → escrita (supertipos)

// leitura
List<? extends Number>

// escrita
List<? super Integer>

🔷 3. equals() e hashCode()

Usados para comparar objetos.

  • equals → comparação de conteúdo
  • hashCode → código numérico do objeto

Regra importante:

Se dois objetos são iguais → devem ter o mesmo hashCode


@Override
public boolean equals(Object obj) {
    if (this == obj) return true;
    if (!(obj instanceof Produto)) return false;

    Produto other = (Produto) obj;
    return nome.equals(other.nome);
}

@Override
public int hashCode() {
    return nome.hashCode();
}
MétodoExplicação
equals()Compara conteúdo
hashCode()Gera código para comparação rápida

🔷 4. Set

Não permite elementos repetidos.


Set<String> set = new HashSet<>();

Tipos:

  • HashSet → rápido, não ordenado
  • TreeSet → ordenado, mais lento
  • LinkedHashSet → mantém ordem de inserção

Métodos:


set.add("A");
set.remove("A");
set.contains("A");
set.size();
set.clear();
MétodoExplicação
add()Adiciona elemento
remove()Remove elemento
contains()Verifica existência
size()Quantidade
clear()Limpa tudo

Operações:


set1.addAll(set2);      // união
set1.retainAll(set2);   // interseção
set1.removeAll(set2);   // diferença
OperaçãoExplicação
addAllUne conjuntos
retainAllInterseção
removeAllDiferença

set.removeIf(x -> x.startsWith("T"));

🔷 5. Map<K,V>

Coleção de chave → valor.

  • Não permite chave duplicada
  • Acesso rápido

Map<String, Integer> map = new HashMap<>();

Tipos:

  • HashMap → rápido
  • TreeMap → ordenado
  • LinkedHashMap → mantém ordem

Métodos:


map.put("A", 10);
map.get("A");
map.remove("A");
map.containsKey("A");
map.containsValue(10);
map.size();
map.clear();
MétodoExplicação
put()Adiciona ou atualiza
get()Busca valor
remove()Remove chave
containsKey()Verifica chave
containsValue()Verifica valor
size()Quantidade
clear()Limpa Map

Navegação:


for (String key : map.keySet()) {
    System.out.println(key);
}

for (Integer value : map.values()) {
    System.out.println(value);
}

for (Map.Entry<String, Integer> entry : map.entrySet()) {
    System.out.println(entry.getKey() + " - " + entry.getValue());
}
FormaExplicação
keySet()Retorna chaves
values()Retorna valores
entrySet()Chave + valor
