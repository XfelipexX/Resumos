🔷 Tabela: Interface vs Default vs Abstract

Conceito Descrição Implementação Observações
Interface Define um contrato que outras classes devem seguir 

interface Animal {
    void fazerSom();
}
Não pode ter construtor
Métodos são públicos por padrão
Método Abstract Método sem corpo (sem implementação) 

abstract class Animal {
    abstract void fazerSom();
}
Obrigatório implementar na subclasse
Pode estar em classe abstrata ou interface
Método Default Método com implementação dentro da interface 

interface Animal {
    default void dormir() {
        System.out.println("Dormindo...");
    }
}
Não é obrigatório sobrescrever
Evita quebrar código antigo
