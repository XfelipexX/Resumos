Comparator

em lambda

Usado para comparar preços -> list.sort((p1, p2) -> p1.getPrice().compareTo(p2.getPrice()));

usado para comparar nomes -> list.sort((s1, s2) -> s1.toUppercase().compareTo(s2.toUppercase()));

Explicação adicional:

Comparator permite criar regras personalizadas de ordenação.

Retorno do compare:

  • Negativo → primeiro menor
  • Zero → iguais
  • Positivo → segundo menor

// crescente
list.sort((p1, p2) -> p1.getPrice().compareTo(p2.getPrice()));

// decrescente
list.sort((p1, p2) -> p2.getPrice().compareTo(p1.getPrice()));

Forma moderna:


list.sort(Comparator.comparing(Produto::getPrice));

Predicate

Documentação

Um boolean

em lambda

Um exemplo de um Predicate que acha valores maior q 100 ->Predicate< tipo> apelido = p -> p.getPrice() >= 100

Explicação adicional:

Predicate representa uma condição lógica.


Predicate<Produto> caro = p -> p.getPrice() > 100;

list.stream()
    .filter(caro)
    .forEach(System.out::println);

Composição:


Predicate<Produto> caro = p -> p.getPrice() > 100;
Predicate<Produto> nomeA = p -> p.getName().startsWith("A");

list.stream()
    .filter(caro.and(nomeA))
    .forEach(System.out::println);

Consumer

Documentação

Consumer ele executa uma função mas nao retorna nada

Um exemplo de um Consumer que add 10% em cada preço de uma lista ->Consumer< tipo> apelido = p -> p.getPrice(p.getPrice * 1.1)

Explicação adicional:

Consumer altera dados (efeito colateral).


list.forEach(p -> p.setPrice(p.getPrice() * 1.1));

Encadeamento:


Consumer<Produto> aumento = p -> p.setPrice(p.getPrice() * 1.1);
Consumer<Produto> print = p -> System.out.println(p);

list.forEach(aumento.andThen(print));

function

Documentação

É uma função que recebe um t e retorna um r

exemplo de um function que transforma a primeira letra de cada nome em uma letra maiuscula (o function esta dentro do .map)

List< String> list = list.stream().map(new toUppercaseName()).collect(collectors.toList)

Explicação adicional:

Function transforma dados.


list.stream()
    .map(nome -> nome.toUpperCase())
    .forEach(System.out::println);

Encadeamento:


Function<Produto, String> nome = p -> p.getName();
Function<String, String> upper = s -> s.toUpperCase();

list.stream()
    .map(nome.andThen(upper))
    .forEach(System.out::println);

Stream

Documentação

É uma sequencia de elementos advinda de uma fonte de dados que oferece suporte a "operações agregadas"

Fonte de dados: coleção, array, função de iteração, recurso de E/S

Explicação adicional:

Stream não altera a coleção original (imutável).

criar uma Steam

usa - Stream< String> stream =

Basta chamar o método stream() ou parallelStream() a partir de qualquer objeto Collection

Stream.of = cria uma stream a partir dos elementos fornecidos como argumentos.

Stream.ofNullable = cria uma stream com um único elemento, a menos que o elemento seja nulo, nesse caso, cria uma stream vazia.

Stream.iterate = cria uma stream onde cada elemento é gerado pela aplicação repetida de uma função ao elemento anterior. Pode ser usado para gerar sequências infinitas.


Stream<String> s = list.stream();
Stream<String> s2 = Stream.of("A","B");

Operações intermediárias

map(Function): Adiciona algo em todas as steam.

filter: Filtra os elementos da stream com base em uma condição especificada pelo predicado.

flatmap: Converte cada elemento em zero ou mais elementos em uma nova stream.

peek: Permite realizar operações de lado (side effects) em cada elemento da stream sem modificar seu conteúdo.

distinct: Remove elementos duplicados da stream.

sorted: Ordena os elementos da stream com base em uma ordem natural ou usando um comparador fornecido.

skip: Pula os primeiros N elementos da stream.

limit: Limita a stream para conter no máximo N elementos.


list.stream()
    .filter(x -> x > 10)
    .map(x -> x * 2)
    .sorted()
    .limit(5)
    .forEach(System.out::println);

Fluxo mental: stream → filter → map → sorted → collect

Operações terminais

removeIf(Predicate): Remove elementos da coleção que atendem à condição especificada pelo predicado.

foreach(Consumer): Executa uma ação para cada elemento da stream.

foreach: Executa uma ação para cada elemento da stream.

forEachOrdered: Executa uma ação para cada elemento da stream respeitando a ordem.

toArray: Converte os elementos da stream em um array.

reduce: Combina os elementos da stream em um único resultado usando uma operação associativa e um identificador.

collect: Realiza uma redução mutável acumulando os elementos da stream em uma coleção ou outra estrutura de dados.

min: Encontra o menor elemento da stream com base em uma ordem natural ou um comparador fornecido.

max: Encontra o maior elemento da stream com base em uma ordem natural ou um comparador fornecido.

count: Retorna o número de elementos na stream.

anyMatch: Verifica se pelo menos um elemento da stream atende à condição especificada.

allMatch: Verifica se todos os elementos da stream atendem à condição especificada.

noneMatch: Verifica se nenhum elemento da stream atende à condição especificada.

findFirst: Encontra o primeiro elemento da stream.

findAny: Encontra qualquer elemento da stream. Em streams paralelas, isso pode ser mais eficiente do que findFirst.


int soma = list.stream().reduce(0, Integer::sum);
long count = list.stream().count();
boolean existe = list.stream().anyMatch(x -> x > 100);

Importante: operações terminais encerram a stream.
